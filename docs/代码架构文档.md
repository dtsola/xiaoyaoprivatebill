## 1. 架构概述

### 1.1 整体架构风格

本项目采用**前后端分离**的分层架构：

```
┌─────────────────────────────────────────────────────────────┐
│                      表现层 (Presentation)                   │
│                      Vue 3 + Vite 前端应用                   │
└─────────────────────────────────────────────────────────────┘
                              │
                              │ HTTP/JSON API
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                       API 路由层 (Routing)                   │
│                       Flask Blueprint 路由                   │
└─────────────────────────────────────────────────────────────┘
                              │
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      业务逻辑层 (Service)                    │
│              数据服务 | 分析服务 | 文件服务 | 会话服务        │
└─────────────────────────────────────────────────────────────┘
                              │
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      数据处理层 (Parser)                     │
│                    支付宝解析器 | 微信解析器                  │
└─────────────────────────────────────────────────────────────┘
                              │
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      数据存储层 (Storage)                    │
│                    本地文件系统 | Pandas DataFrame           │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 架构原则

| 原则 | 说明 | 实践 |
|------|------|------|
| **分层解耦** | 各层职责清晰，通过接口通信 | API 层不处理业务逻辑 |
| **模块化** | 功能按模块拆分，便于维护 | 每个功能独立模块 |
| **本地优先** | 数据不上传，保护隐私 | 所有数据本地处理 |
| **向后兼容** | API 接口保持稳定 | 重构已完成，API 完全兼容 |

### 2.2 模块职责详解

#### 2.2.1 API 路由层 (`api/`)

**职责**：处理 HTTP 请求，调用业务逻辑层，返回 JSON 响应。

**模块说明**：

| 模块 | 路由前缀 | 主要功能 |
|------|---------|---------|
| `analysis.py` | `/api/` | 年度/月度分析、分类统计、时间分析 |
| `transactions.py` | `/api/transactions` | 交易记录查询、筛选、分页 |
| `files.py` | `/api/files` | 文件上传、列表、删除、清空 |
| `session.py` | `/api/session`, `/api/demo` | 演示模式会话管理 |

**代码示例**：
```python
# api/analysis.py
from flask import Blueprint, request, jsonify
from services.analysis_service import AnalysisService

analysis_bp = Blueprint('analysis', __name__)

@analysis_bp.route('/api/yearly_analysis', methods=['GET'])
def get_yearly_analysis():
    """获取年度分析数据"""
    year = request.args.get('year', type=int)
    if not year:
        return jsonify({'success': False, 'error': '年份参数必填'}), 400

    service = AnalysisService()
    result = service.get_yearly_analysis(year)
    return jsonify({'success': True, 'data': result})
```

#### 2.2.2 业务逻辑层 (`services/`)

**职责**：封装核心业务逻辑，处理数据计算、转换、聚合。

**模块说明**：

| 模块 | 职责 |
|------|------|
| `data_service.py` | 管理交易数据缓存，提供数据查询接口 |
| `analysis_service.py` | 实现各种统计分析算法（年度、月度、分类等） |
| `file_service.py` | 处理文件上传、调用解析器、管理文件生命周期 |

**代码示例**：
```python
# services/analysis_service.py
from services.data_service import DataService

class AnalysisService:
    def __init__(self):
        self.data_service = DataService()

    def get_yearly_analysis(self, year: int) -> dict:
        """获取年度分析数据"""
        df = self.data_service.get_transactions_by_year(year)
        if df.empty:
            return self._empty_yearly_result(year)

        return {
            'year': year,
            'total_income': float(df[df['amount'] > 0]['amount'].sum()),
            'total_expense': float(abs(df[df['amount'] < 0]['amount'].sum())),
            'monthly_data': self._calculate_monthly_data(df),
            'category_breakdown': self._calculate_category_breakdown(df)
        }
```

#### 2.2.3 数据解析层 (`parsers/`)

**职责**：解析不同格式的账单文件，输出统一格式的 DataFrame。

**统一数据格式**：
```python
{
    'date': datetime,        # 交易时间
    'amount': float64,       # 交易金额（负数为支出）
    'category': string,      # 消费分类
    'pay_method': string,    # 支付方式（支付宝/微信）
    'shop_name': string,     # 商户名称
    'description': string,   # 交易说明
    'type': string          # 交易类型（支出/收入/转账）
}
```

**模块说明**：

| 模块 | 支持格式 | 解析方式 |
|------|---------|---------|
| `alipay.py` | CSV | Pandas read_csv，编码检测 |
| `wechat.py` | CSV, XLSX | Pandas read_csv/read_excel，OpenPyXL |

**代码示例**：
```python
# parsers/alipay.py
import pandas as pd
import chardet

class AlipayParser:
    @staticmethod
    def parse(file_path: str) -> pd.DataFrame:
        """解析支付宝 CSV 账单"""
        # 检测文件编码
        with open(file_path, 'rb') as f:
            encoding = chardet.detect(f.read())['encoding']

        # 读取并标准化列名
        df = pd.read_csv(file_path, encoding=encoding, skiprows=16)
        df = AlipayParser._standardize_columns(df)
        df = AlipayParser._normalize_data(df)

        return df

    @staticmethod
    def _standardize_columns(df: pd.DataFrame) -> pd.DataFrame:
        """将支付宝列名映射为统一格式"""
        column_map = {
            '交易时间': 'date',
            '商品说明': 'description',
            '收/支': 'type',
            '金额': 'amount',
            # ... 更多映射
        }
        return df.rename(columns=column_map)
```

### 3.2 模块职责详解

#### 3.2.1 API 客户端层 (`src/api/`)

**职责**：封装所有后端 API 调用，统一请求/响应处理。

**基础配置**：
```javascript
// src/api/index.js
const BASE_URL = import.meta.env.VITE_API_BASE_URL || '/api'

const apiClient = {
    async request(endpoint, options = {}) {
        const url = `${BASE_URL}${endpoint}`
        const response = await fetch(url, {
            headers: {
                'Content-Type': 'application/json',
                ...options.headers
            },
            ...options
        })

        if (!response.ok) {
            throw new Error(`HTTP ${response.status}`)
        }

        return response.json()
    }
}

export default apiClient
```

**API 模块示例**：
```javascript
// src/api/analysis.js
import apiClient from './index'

export const analysisApi = {
    // 获取年度分析
    getYearlyAnalysis(year) {
        return apiClient.request(`/yearly_analysis?year=${year}`)
    },

    // 获取月度分析
    getMonthlyAnalysis(year, month) {
        return apiClient.request(`/monthly_analysis?year=${year}&month=${month}`)
    },

    // 获取分类消费
    getCategoryExpenses(year, category = null) {
        const params = new URLSearchParams({ year })
        if (category) params.append('category', category)
        return apiClient.request(`/category_expenses?${params}`)
    }
}
```

#### 3.2.3 状态管理层 (`src/stores/`)

**职责**：管理全局状态，处理数据缓存和异步操作。

**Store 示例**：
```javascript
// src/stores/data.js
import { defineStore } from 'pinia'
import { analysisApi } from '@/api/analysis'

export const useDataStore = defineStore('data', {
    state: () => ({
        yearlySummary: null,
        monthlyData: [],
        categoryData: [],
        availableYears: [],
        loading: false,
        error: null
    }),

    actions: {
        async loadYearlyAnalysis(year) {
            this.loading = true
            this.error = null
            try {
                const response = await analysisApi.getYearlyAnalysis(year)
                if (response.success) {
                    this.yearlySummary = response.data
                    this.monthlyData = response.data.monthly_data
                    this.categoryData = response.data.category_breakdown
                }
            } catch (err) {
                this.error = err.message
            } finally {
                this.loading = false
            }
        }
    }
})
```

## 6. 核心流程

### 6.1 应用启动流程

**后端启动**：
```python
# app.py
from flask import Flask
from api import analysis_bp, files_bp, session_bp, transactions_bp
from config import Config

def create_app():
    app = Flask(__name__)
    app.config.from_object(Config)

    # 注册蓝图
    app.register_blueprint(analysis_bp)
    app.register_blueprint(files_bp)
    app.register_blueprint(session_bp)
    app.register_blueprint(transactions_bp)

    # 注册静态文件服务（生产环境）
    if not app.debug:
        @app.route('/')
        def index():
            return send_from_directory('frontend/dist', 'index.html')

    return app

if __name__ == '__main__':
    app = create_app()
    app.run(host='0.0.0.0', port=8080, debug=True)
```

**前端启动**：
```javascript
// main.js
import { createApp } from 'vue'

app.use(router)          // 路由

app.mount('#app')
```
