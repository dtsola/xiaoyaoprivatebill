最近想整理自己的支付宝/微信账单，发现市面上的工具都需要上传数据到服务器，
担心隐私泄露，就自己动手做了一个。

---

**演示视频**（90秒快速了解）
https://www.bilibili.com/video/BVxxxxxxxxxxxxx

**GitHub 仓库**
https://github.com/dtsola/xiaoyaoprivatebill

---

### 核心特点

- 🔒 **隐私安全** - 数据完全本地处理，不上传任何服务器
- 🔐 **手动清除** - 随时手动清除账单数据，完全掌控您的财务隐私
- 📊 **多维度分析** - 年度、月度、分类、时间、消费洞察
- 📁 **多格式支持** - 支付宝 CSV、微信 CSV/XLSX 账单文件
- 🚀 **快速部署** - Docker 一键启动，开箱即用

### 功能预览

![上传账单](docs/产品文档/产品截图/08-上传账单.png)
拖拽上传，自动识别支付宝/微信账单格式

![年度总览](docs/产品文档/产品截图/02-年度总览.png)
年度收支总览，趋势一目了然

![月度分析](docs/产品文档/产品截图/03-月度分析.png)
月度收支趋势，环比同比分析

![分类分析](docs/产品文档/产品截图/04-分类分析.png)
分类占比分析，钱花在哪一看便知

![消费洞察](docs/产品文档/产品截图/06-消费洞察-01.png)
智能分析消费习惯，发现省钱机会

### 技术栈

- 前端：Vue 3 + Vite + ECharts
- 后端：Flask + Pandas + NumPy
- 部署：Docker + Nginx

### 快速体验

```bash
# 克隆项目
git clone https://github.com/dtsola/xiaoyaoprivatebill.git
cd xiaoyaoprivatebill

# 一键启动
docker-compose up -d

# 访问应用
# 浏览器打开: http://localhost:8888
```
