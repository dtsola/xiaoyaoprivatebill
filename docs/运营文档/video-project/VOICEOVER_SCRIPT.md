## 纯文本版（无停顿提示）

想要分析自己的消费账单，但不敢用第三方工具？因为担心数据上传、隐私泄露...

所以我开发了这个工具——小遥账单助手。一个完全本地化的账单分析工具。数据不上传服务器，支持支付宝和微信账单。

拖拽上传，自动识别格式。支付宝 CSV、微信 CSV/XLSX 都支持。

年度总览一目了然，收入支出清晰可见。

所有数据都在本地处理，随时手动清除，隐私无忧。

前端使用 Vue 3 加 ECharts，后端使用 Flask 加 Pandas。一行 Docker 命令即可启动。

项目已在 GitHub 开源，完全免费使用。如果你也在寻找一个隐私安全的账单分析方案，欢迎试用并给个Star支持。

## 录音技巧

| 要点 | 说明 |
|------|------|
| **语速** | 适中，每秒约3-4个字 |
| **语调** | 友好、真诚，不生硬 |
| **音量** | 统一，避免忽大忽小 |
| **环境** | 安静房间，无回声 |
| **格式** | MP3，单声道，128kbps |
| **采样率** | 44.1kHz |

## 推荐录音软件

1. **Windows**: Audacity（免费）或 系统录音机
2. **Mac**: GarageBand 或 QuickTime Player
3. **手机**: 语音备忘录（足够使用）

---

## 后期处理建议

1. **去除噪音** - 使用 Audacity 的噪音消除功能
2. **调整音量** - 统一音量到 -6dB
3. **导出格式** - MP3, 128kbps, 44.1kHz, 单声道

导出文件名: `voiceover.mp3`
放置位置: `docs/运营文档/video-project/public/audio/voiceover.mp3`
