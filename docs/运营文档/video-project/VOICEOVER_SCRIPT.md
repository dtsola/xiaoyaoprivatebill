## 录音技巧

| 要点 | 说明 |
|------|------|
| **语速** | 适中，每秒约3-4个字 |
| **语调** | 友好、真诚，不生硬 |
| **音量** | 统一，避免忽大忽小 |
| **环境** | 安静房间，无回声 |
| **格式** | MP3，单声道，128kbps |
| **采样率** | 44.1kHz |

## 推荐录音软件

1. **Windows**: Audacity（免费）或 系统录音机
2. **Mac**: GarageBand 或 QuickTime Player
3. **手机**: 语音备忘录（足够使用）

---

## 后期处理建议

1. **去除噪音** - 使用 Audacity 的噪音消除功能
2. **调整音量** - 统一音量到 -6dB
3. **导出格式** - MP3, 128kbps, 44.1kHz, 单声道

导出文件名: `voiceover.mp3`
放置位置: `docs/运营文档/video-project/public/audio/voiceover.mp3`
