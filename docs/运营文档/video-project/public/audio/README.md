## 录音要求

### 技术参数
- **格式**: MP3
- **比特率**: 128kbps
- **采样率**: 44.1kHz
- **声道**: 单声道
- **音量**: -6dB（录音后统一调整）

### 录音技巧
- **语速**: 适中，每秒约 3-4 个字
- **语调**:
  - 开场：有共鸣感，提出问题
  - 介绍：自信、清晰
  - 功能展示：轻快、流畅
  - 结尾：热情、邀请
- **音量**: 统一，避免忽大忽小
- **环境**: 安静房间，无回声

### 推荐录音软件
- **Windows**: Audacity（免费）
- **Mac**: GarageBand
- **手机**: 语音备忘录
